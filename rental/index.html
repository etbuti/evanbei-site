East / West 租赁运行标准（Operating Standard）

目的：减少反复解释、统一执行口径、稳定现金流、降低风险。
Purpose: consistency, stability, compliance, and reduced friction.

总原则 / Operating Principle

两套都可以挂牌，但在同一时间只需要成交其中一套也完全正常；另一套作为缓冲位（溢价定位/选择性出租）。
Both properties may be listed, while it is acceptable that only one is actively let at a time. The other is a deliberate buffer (premium/selective positioning).
East 价高自营 · 精选租客 / Premium self-managed West 中介托管 · 标准服务 / Agent-managed standard
East → Track A（价高自营：格式操作法 + 精选租客 + 服务溢价） West → Track B（中介托管：标准定价 + 标准服务 + 现金流稳定） 核心：切换是“运营动作”，不是“情绪动作”。 Core: switching is operational, not emotional.

  • Start next month / 下个月启动

East（Track A）— 价高自营，优质优价 / Premium Self-Managed

Selective

目标：溢价来自“清晰规则 + 快速响应 + 主动选择优质租客”。
Objective: premium through clarity, responsiveness, and selective letting.

申请 Application → 筛选 Screening → 资信 References/Right-to-Rent → 签约 Agreement → 服务 Service
  • 合租溢价 / No sharers；转租溢价 / No subletting；短租溢价 / No short-term letting
  • 签约前先发 House Rules（让规则先落地）
  • 维修响应目标：常规问题 24–48 小时内给出安排
  • 服务溢价必须“可感知”：干净、稳定、响应、规则清晰
East 快速清单 / Quick checklist
  • 申请表 / Application form
  • Right to Rent
  • Employment + landlord references
  • Deposit scheme + inventory evidence
  • House Rules written acknowledgement

West（Track B）— 中介托管 / Agent-Managed Standard

Standard

目标：标准化、低参与、快成交、稳现金流。
Objective: standardisation, minimal involvement, stable cash-flow.

  • 标准定价 / Market rent：第 14 天复盘，第 28 天必须调整
  • 空置不收管理费 / No management fee during voids
  • 维修授权 ≤ £50 中介直接处理；超过需书面同意
  • 中介承担租客沟通与日常管理，房东不参与
West 空置止损节奏 / Void-control cadence
  • Day 14：无带看 = 价格/照片/广告问题
  • Day 28：必须切到成交价（中位）或给激励
  • Day 45：复盘中介执行力；必要时换中介

切换逻辑 / Switching Logic

切换是运营动作，不是情绪动作。Switching is operational, not emotional.

切换到 East / Switch to East when

  • 需要更强控制权（租客筛选/价格保护）
  • 需要保留缓冲位（可住/可调）
  • 市场对“溢价定位”给得出价

切换到 West / Switch to West when

  • 要稳定现金流、少操心
  • 精力要回收，不想参与日常管理
  • 市场适合走标准快成交
红线 / Red lines
  • 不要“挂牌却拒绝带看” / Don’t list as available while refusing viewings
  • 不要长期挂“明显不可能成交”的价格 / Don’t keep an impossibly high price without review
  • 不要混账（两套/两名义必须分开）/ Don’t mix accounts or records
  • 广告与筛选不得使用歧视性语言 / No discriminatory language
